# Amazon-Prime
This Power BI dashboard provides a comprehensive analysis of Amazon Prime content and performance using interactive visualizations and key metrics. The goal of this dashboard is to help users understand content distribution, trends, and viewer preferences across the platform.
Key Insights Covered

Total number of movies and TV shows available on Amazon Prime

Content distribution by genre, country, and release year

Movies vs TV shows comparison

Ratings and maturity-level analysis

Trend analysis of content releases over time
